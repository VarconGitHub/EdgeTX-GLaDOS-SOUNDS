# EdgeTX-GLaDOS-SOUNDS
GLaDOS sounds synth from uberduck.ai and 15.ai

File format:
- File Name: 123456.wav (up to 6 characters plus .wav)
- Sample Rate: 32 kHz (or 16 Khz, 8Khz)
- Bits / Sample: 16 (or 8)
- Tracks: 1, mono.
- Compression Codec: PCM (or u-law, a-law)
